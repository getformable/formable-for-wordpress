Integrates Formable into your WordPress install. This allows you to add your Formable form in WordPress.

== Description ==
This plugin connects your WordPress install with [Formable](https://www.getformable.com). Which makes getting forms from Formable in WordPress super easy!

You'll give WordPress permission to include Formable forms in your posts and pages.

[Read more here](https://www.getformable.com/support/wordpress-plugin)

Even if you don't connect to Formable, you can add the shortcodes for Formable. To add a form to a post or page use: `[formable id="your-forms-id"]`

== Installation ==
1. Upload "formable" folder to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" screen in WordPress.
1. Connect to your Formable account on the "Formable" menu item.
5. Give your WordPress install permission to read your account information.
6. Party!

== Frequently Asked Questions ==

= Is Formable free? =
Yes. You can sign up for a free account on [https://app.getformable.com/sign-up](https://app.getformable.com/sign-up) or in the plugin.

= Will this plugin read my messages? =
No. This plugin only reads which Forms you've made.
